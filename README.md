# EmuHub

A multi-console emulator game launcher that serves as a hub for all your video games.

## Emulator and File Support

| Console  | Emulator | File Extension |
| -------- | -------- | -------------- |
| Switch   | Yuzu     | .nsp           |
| Wii U    | Cemu     | .rpx           |
| Wii      | Dolphin  | .iso, .wbfs    |
| GameCube | Dolphin  | .iso, .gcm     |
| 3DS      | Citra    | .3ds, .cci     |
| DS       | DeSmuME  | .zip, .nds     |
| GBA      | mGBA     | .zip, .gba     |
| PSP      | PPSSPP   | .iso           |
| PS2      | PCSX2    | .iso           |
