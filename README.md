# EmuHub
A console emulator game launcher that serves as a hub for all your video games.

## Emulator and File Support
| Console          | Emulator      | File Extension |
| ---------------- | ------------- | ---------- |
| Switch  | Yuzu  | .nsp |
| Wii U  | Cemu  |  .rpx |
| Wii |   Dolphin | .iso, .wbfs |
| GameCube  | Dolphin | .iso, .gcm |
| 3DS | Citra | .3ds, .cci |
| GBA | mGBA  | .zip, .gba |
| PSP | PPSSPP  | .iso |
